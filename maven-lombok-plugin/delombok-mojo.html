lombok:delombok

Full name:

org.projectlombok:maven-lombok-plugin:0.9.3-SNAPSHOT:delombok

Description:

Delombok java source with lombok annotations.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-sources.

Required Parameters

Name Type Since Description
encoding String - Encoding.
Default value is: ${project.build.sourceEncoding}.
outputDirectory File - Location of the generated source files.
Default value is: ${project.build.directory}/generated-sources/delombok.
skip boolean - Specifies whether the delombok generation should be skipped.
Default value is: false.
sourceDirectory File - Location of the lombok annotated source files.
Default value is: ${project.build.directory}/../src/main/lombok.
verbose boolean - Verbose flag.
Default value is: false.

Parameter Details

encoding:

Encoding.
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${lombok.encoding}
  • Default: ${project.build.sourceEncoding}

outputDirectory:

Location of the generated source files.
  • Type: java.io.File
  • Required: Yes
  • Expression: ${lombok.outputDirectory}
  • Default: ${project.build.directory}/generated-sources/delombok

skip:

Specifies whether the delombok generation should be skipped.
  • Type: boolean
  • Required: Yes
  • Expression: ${lombok.delombok.skip}
  • Default: false

sourceDirectory:

Location of the lombok annotated source files.
  • Type: java.io.File
  • Required: Yes
  • Expression: ${lombok.sourceDirectory}
  • Default: ${project.build.directory}/../src/main/lombok

verbose:

Verbose flag.
  • Type: boolean
  • Required: Yes
  • Expression: ${lombok.verbose}
  • Default: false
